# 联邦学习后门攻击检测

一种基于Transformer序列建模的无监督联邦学习后门攻击检测框架，该方法通过深度分析神经网络多层激活模式的时序依赖关系来识别潜在的恶意客户端模型。

## 快速开始

```bash
# 1. 安装依赖
chmod +x setup.sh
./setup.sh

# 2. 运行实验  
python main.py
```

## 项目结构

- `config.py`: 实验配置参数
- `main.py`: 主实验脚本
- `data/`: 数据加载和联邦分割
- `models/`: ResNet-20模型
- `attack/`: Pattern后门攻击
- `federated/`: 联邦学习实现
- `feature/`: 特征提取
- `detection/`: 检测分类器
- `utils/`: 工具函数

## 系统要求

- Python 3.8+
- PyTorch 1.12+
- CUDA 11.0+ (可选)
- 8GB+ RAM
